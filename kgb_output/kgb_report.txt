INTERPRÉTATION
----------------------------------------
KGBf  : détecte les anomalies dans l'espace des PC dominantes.
        Efficace si le botnet génère un comportement très différent
        du trafic normal dans les directions de plus grande variance.

KGBfog: détecte les anomalies dans le 'bruit' résiduel (PC mineures).
        Efficace pour les botnets discrets qui se fondent dans le trafic
        normal sur les features dominantes.

Scénario 8 (Murlo) : botnet avec C&C propriétaire, scans DCERPC.
Entropies faibles → comportement répétitif et peu diversifié.
